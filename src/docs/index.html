Brainly style guide

Version {{ site.data.package.version }}

Welcome to the official Brainly style guide! This style guide is being used across all language versions of the Brainly website (e.g. brainly.com, znanija.com, nosdevoirs.fr), on our corporate website (brainly.co) and some smaller sites (e.g. landing pages).

How to use it?

In order to use style guide components, you have to include CSS file in the <head> of your website:

  <link href="https://styleguide.brainly.com/{{ site.data.package.version }}/style-guide.css" rel="stylesheet"/>

By default icons will not work on older browsers. To fix that you can include following script just before the ending </body> tag:

  <script src="https://styleguide.brainly.com/%%images/external-svg-icons-polyfill.js%%"></script>
